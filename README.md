# Global Events Platform

A modern event discovery web application built with **Next.js**.  
The platform allows users to explore events happening in major cities around the world such as **Tokyo, San Francisco, and Milan**.

Users can browse event categories, view detailed event pages, and register their email to stay updated about upcoming events.

---

## 🚀 Features

- 🌆 Browse events by city
- 📅 View detailed event information
- 📧 Register email for event updates
- ⚡ Dynamic routing with Next.js
- 🖼️ Event images powered by Unsplash
- 🎨 Responsive UI with Sass styling

---

## 🗺️ Available Pages

| Page | Route | Description |
|-----|------|-------------|
| Home | `/` | Main landing page with featured cities |
| Events | `/events` | Lists all event categories |
| Category Page | `/events/[city]` | Shows events in a specific city |
| Event Page | `/events/[city]/[id]` | Displays detailed event information |
| About | `/about-us` | Information about the platform |
